# LRS戦略 拡張定義（2026確定版）
> 基本ルールは `Node/Rules_v2026.md` を参照
> 本ファイルは Rules_v2026.md に含まれない拡張概念を格納

---

## 二層運用（固定仕様）

### 共通原理
- 戦略はLRSのみ（増やさない）
- ①で方向確定、④の水平線表示、⑤で実行
- ロジックは全時間軸で同一、役割のみ分離

### レイヤー①：M15 → M1（回収レイヤー）
| 項目 | 内容 |
|------|------|
| 目的 | 頻度・検証母数・おこぼれ回収 |
| 環境 | M15で①が明確 |
| 実行 | M1で④→⑤ |
| RR | 1:1固定 |
| 特性 | 取り逃し許容・感情を切りやすい |

### レイヤー②：H4 → M15（主砲レイヤー）
| 項目 | 内容 |
|------|------|
| 目的 | 期待値・伸ばし・トレンドフォロー |
| 環境 | H4のネックライン・押し安値/戻り高値・レンジ端・明確な水平 |
| 条件 | H4で反発/抑制確認 → M15で④形成 |
| 実行 | ⑤ |
| RR | 1:1以上（伸ばし前提） |

---

## ①〜⑤フェーズ定義

| # | 名称 | 定義 |
|---|------|------|
| 〇 | 場 | 初期BOXのみ（未発生。表示なし） |
| ① | R1 | M15でのBOXブレイク → 方向完全確定 |
| ② | R2 | ①後の戻り（①起点の高安値は未更新） |
| ③ | R3初動 | ①方向への再挑戦 |
| ④ | R2-3 | ③への本気の反抗 → **④起点に水平線を引く** |
| ⑤ | R3本命 | ④水平線の明確な敗北 → **実行** |

### 表示ルール
- ①の方向矢印と④の水平線のみ。この2要素で完結。

---

## 波の生成原理（超重要）

- 上位概念波は、常に下位概念波の「転換（S1）」から生まれる
- 「直前の片波S波」を崩したかどうかが本質
- R1/R2/R3の番号は便宜的。本質は「Rの片波 ＝ 下位S波を崩して発生した新規S1」

### S1転換の方向判定（絶対固定）
| S1の方向 | 該当フェーズ |
|---|---|
| 基準波と**同方向** | R1(①)・R3初動(③)・R3(⑤) |
| 基準波と**逆方向** | R2(②)・R2-3(④) |

---

## R2-3の定義

- M1でS1転換波発生 → 「R2は発生した」と判断
- R2-3は必ずしも高値・安値を更新しない
- 本質は「拮抗・レンジ・縮小・もみ合い」が発生し、R3初動が止められなかったこと

### 水平線ルール（超重要）
- 水平線 ＝ R2-3発生が観測された事実の固定
- 水平線を表示している ＝ R2-3は既に発生している
- M1でR2-3起点（S1）を判断した瞬間に水平線を引く

---

## ①R1の追加フィルター

- ①は「動いたから①」ではない
- **L波（H4）の意味あるライン反発を起点としてのみ①を認める**
- この定義を満たさないM15の動きは①として扱わない

---

## R1の時間的条件

- M15のR1は**ローソク足10本未満（目安3〜8本）**の短命な片波のみが主対象
- この条件下でR2・R2-3が明確に発生しやすく、M1のS波と時間同期が取りやすい

---

## トリガー線ルール

| 線の種類 | 引き方 | 用途 |
|---|---|---|
| ④トリガー | ④起点から**水平に右へ画面いっぱい** | **実行判断** |
| 斜めトレンドライン | 通常通り | 環境認識用 |
| 水平線（MT5自動） | 全価格帯 | 環境認識用 |

## 核心サマリー

> 「R1から発生する短期S3を1:1で狙う」= LRS戦略の絶対的核心

> 「1波は仮定、3波は回収。結果が出て初めてカウントが確定する。」
